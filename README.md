# G.pto

A mobile-first self-hosted cloud browser.

## Vision

g.pto is an open-source project focused on providing a lightweight, low-latency browser streaming experience designed primarily for Android devices.

Unlike traditional remote desktop solutions, g.pto focuses on streaming only the browser with an interface optimized for mobile users.

## Project Goals

- Mobile-first user interface
- Low latency
- Lightweight Docker image
- Chromium-based browser
- WebRTC streaming
- Easy self-hosting
- Open source
